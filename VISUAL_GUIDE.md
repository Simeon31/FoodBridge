# FoodBridge - Visual Guide & UI Preview ??

## Application Screenshots & Layout Guide

This document describes the visual appearance and layout of the FoodBridge application after Tailwind admin dashboard integration.

## Color Scheme

### Primary Colors
- **Primary Blue:** `#3c50e0` - Main brand color (buttons, links, active states)
- **Secondary Blue:** `#80caee` - Accents and highlights
- **Background:** `#f1f5f9` - Light gray for page background
- **White:** `#ffffff` - Card and panel backgrounds

### Status Colors
- **Success Green:** `#10b981` - Success messages, positive metrics
- **Danger Red:** `#ef4444` - Error messages, negative alerts
- **Warning Orange:** `#f59e0b` - Warning messages, pending states
- **Info Blue:** `#259ae6` - Informational messages

### Text Colors
- **Primary Text:** `#1f2937` - Main content text
- **Secondary Text:** `#64748b` - Labels, descriptions
- **Light Text:** `#9ca3af` - Placeholders, disabled text

### Dark Mode (Ready)
- **Background:** `#1a222c` - Dark background
- **Panel:** `#24303f` - Dark cards
- **Text:** `#e5e7eb` - Light text on dark

## Component Specifications

### Typography Scale
- **Page Title (H1):** 26px, Bold
- **Card Title (H3):** 20px, Semibold
- **Stat Number:** 24px, Bold
- **Body Text:** 16px, Regular
- **Small Text:** 14px, Regular
- **Tiny Text:** 12px, Regular

### Spacing
- **Container Padding:** 16px mobile, 24px desktop
- **Card Padding:** 30px
- **Element Gaps:** 16-24px
- **Section Margins:** 24-32px

### Border Radius
- **Cards:** 8px
- **Buttons:** 8px
- **Inputs:** 8px
- **Icon Circles:** 50% (full circle)
- **Badges:** 4px

### Shadows
- **Cards:** `0px 1px 3px rgba(0, 0, 0, 0.12)`
- **Header:** `0px 4px 10px rgba(0, 0, 0, 0.12)`
- **Hover:** Slightly elevated shadow

### Animations
- **Sidebar Toggle:** 300ms ease-in-out
- **Button Hover:** 200ms ease
- **Loading Spinner:** 1s linear infinite
- **Notification Ping:** 5s cubic-bezier

- Sidebar: Drawer (hidden by default)
- Stats: 1 column
- Header: Compact with hamburger
- Typography: Slightly smaller

### Tablet (640px - 1024px)
- Sidebar: Drawer (overlay)
- Stats: 2 columns
- Header: More space, search visible

### Desktop (? 1024px)
- Sidebar: Fixed, collapsible
- Stats: 4 columns
- Header: Full features, search bar

## Icon Library

All icons are inline SVG for:
- Better performance
- Easy color changes
- Scalability
- No external dependencies

### Icon Set Used:
- Dashboard: Chart/Grid icon
- Donations: Money/Dollar icon
- Requests: Clipboard/Document icon
- Organizations: Building/People icon
- Profile: User icon
- Notification: Bell icon
- Search: Magnifying glass icon
- Menu: Hamburger (3 lines)
- Logout: Arrow exit icon
- Success: Checkmark
- Warning: Exclamation
- Error: X mark
- Info: Info circle
